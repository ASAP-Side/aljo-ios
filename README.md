# 알죠앱 - iOS

## 사용법
1. Install Tuist
   
    ```bash
    curl -Ls https://install.tuist.io | bash
    ```

2. Tuist Singing 데이터 추가

3. Update Dependencies
    ```bash
    tuist fetch
    ```

    ```bash
    tuist fetch --update
    ```

4. Run Tuist Generate
    ```bash
    tuist generate
    ```
